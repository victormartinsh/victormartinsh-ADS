Introdução a HTML e CSS

  • Para que servem:

    • HTML (Hypertext Markup Language) é usado para criar a estrutura de uma página web, definindo elementos como cabeçalho, parágrafos e links.

    CSS (Cascading Style Sheets) é usado para estilizar esses elementos, definindo cores, tamanhos de fonte e layout.

  • Como são usados:

    • HTML é usado para definir a estrutura de uma página, enquanto o CSS é usado para aplicar estilos e torná-la visualmente atraente.

  • Referências:

    • iMasters: Oferece artigos e tutoriais em português sobre HTML e CSS.

    Tableless: Um blog brasileiro especializado em web design e desenvolvimento.

Tags HTML

  • Para que servem:

    • As tags HTML são usadas para estruturar o conteúdo de uma página web. Cada tag tem uma função específica, como "h1" para cabeçalho de nível 1, "p" para parágrafos e "a" para links.

  • Como são usados:

    • As tags HTML são incorporadas no código da página para definir elementos e sua hierarquia.

  • Referências:

    • Referência HTML do MDN em Português: Uma referência completa das tags HTML em português.

    W3Schools HTML Tutorial em Português: Oferece um tutorial abrangente sobre HTML em português.

    CSS

  • Para que servem:

    • CSS é usado para estilizar páginas web, controlando a aparência visual dos elementos HTML. Isso inclui cores, fontes, margens, entre outros.

  • Como são usados:

    • As regras CSS são definidas no código da página ou em arquivos externos, e elas se aplicam aos elementos HTML usando seletores.

  • Referências:

    W3Schools CSS Tutorial em Português: Oferece um tutorial abrangente sobre CSS em português com exemplos práticos.
